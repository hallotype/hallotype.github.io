The fonts in NES games are usually build up by 8x8 bytes per character. By plotting the binary data, the graphic tiles become visibe and the font is exposed. If you know where the font data is located, you can draw the characters into a UFO and generate a fully working font.

Here is the graphic data set of Super Mario Bros.
In this game the type is monochome, so there is only one layer. Very convenient, the font clearly visible, and extracting is easy.
